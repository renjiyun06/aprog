桌面 OS 形态的 Web 前端

aprog 平台是一台「AI 程序运行的虚拟 OS」, 这里是它的 shell —— 跟 Windows 11 桌面非常接近, 只是 kernel 不同. 浏览器是这台远端 OS 的客户端, 换设备登录看到的是同一张桌面. 下面每个卡片是一张 mockup 截图; 点进去看高保真渲染.
