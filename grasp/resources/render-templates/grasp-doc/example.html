每个包管理器各一层 → 改 npm 不重装 apt。Apt 层尾部 rm /var/lib/apt/lists/* 保持轻量。

Where

linchpin-api/app/sandbox.py:177 — DockerSandbox.ensure_image linchpin-api/app/sandbox.py:225 — _generate_dockerfile linchpin-api/app/models.py:446 — EnvironmentPackages + 名称正则白名单

Caveats

  • --break-system-packages 是 Debian trixie PEP 668 必需，sandbox 内安全。
  • 每个 go pkg 单独 RUN，一个失败不污染其他。
  • 缓存在 Docker daemon 本地。多 host 需要 registry 共享派生镜像。
