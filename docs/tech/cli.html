设计要点

  • 单 binary: 用户不需要 Node / Python / Docker. curl 装就跑.
  • 没本地状态除了凭据: 所有 process 状态在 control plane. CLI 重启不丢东西 (除了 TUI 历史; 但 ring buffer 会补回来).
  • TUI 通过 ticket 鉴权 WS: token 不进 WS URL (会进 nginx 日志). ticket 是 attach response 给的一次性 5 秒 nonce.
