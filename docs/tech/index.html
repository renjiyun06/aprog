关键时序速览

每张卡片缩略图展示主要 actor 之间的消息形状, 点击进入完整时序图.

spawn

注册 process + 初始化快照骨架. 不起沙箱, 不烧 compute.

attach · mode 1

首次进入: 起沙箱 + 注入快照 + 启 driver + 打通流.

attach · mode 3

re-attach: 沙箱在, ring buffer 补 detach 间隙, 再续 live stream.

/exit

收尾: tar-out 沙箱 → 覆盖快照 → 销毁沙箱 → state=exited.

/detach

最轻量: 仅断 UI 通道. 沙箱继续吃 compute, ring buffer 兜事件.

login

argon2id 校验 + 新 AuthSession 入库 + 客户端写 credentials.
