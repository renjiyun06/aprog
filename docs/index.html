aprog 文档

aprog 是一台“运行 AI 程序的虚拟 OS”。程序（program）是装在平台上的技能， 进程（process）是程序的一次运行，沙箱（sandbox）是给它临时算力的环境。 这份文档由浅入深地说明平台的概念、界面、组件、协议与持久化模型。

一句话心智模型

把传统操作系统的词汇平移过来，aprog 的一切都能对上号：

操作系统aprog说明
可执行程序程序 Program一段以 skill 形态存在的能力（SKILL.md），静态实体
进程进程 Process程序的一次运行，可休眠 / 唤醒，有自己的 PID 与目录
/proc/<pid>/进程目录~/.aprog/<pid>/，进程的真正状态，也是 harness 的 cwd（不挂卷，靠检查点持久化）
CPU引擎 Engine底层 agent loop（当前 Claude Agent SDK，可替换）
内存 / 运行环境沙箱 Sandboxper-process 容器，临时算力，由 PPIO 提供
